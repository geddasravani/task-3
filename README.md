# Task 3 – SRAVS Admin Dashboard

## Overview
Custom admin dashboard built on Mazer-inspired Bootstrap 5 layout.

## Features
- Sidebar navigation with active states
- Stats cards loaded from data.json
- Projects table with status badges
- Skill progress bars (animated)
- Message inbox with unread indicators
- Contact form with success feedback
- Fully responsive (mobile sidebar toggle)

## Setup
1. Clone/download this repo
2. Open in VS Code
3. Install Live Server extension
4. Click "Go Live" on index.html

## Changes Made
- Custom sidebar layout replacing Mazer's default
- data.json integration for dynamic content
- CSS variables for consistent theming
- Mobile responsive toggle

## Tech Stack
Bootstrap Icons · Vanilla JS · CSS3 · HTML5
